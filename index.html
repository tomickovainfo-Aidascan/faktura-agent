Automatické stahování faktur
z Gmailu na Google Drive

Krok za krokem: jak si nastavit vlastního agenta, který hlídá příchozí faktury a ukládá je automaticky – bez klikání, bez zapomínání.

Obtížnost: střední Nastavení: 30–45 minut Náklady: zdarma Spouštění: každou hodinu

Co tento agent dělá

Hlídá Gmail každou hodinu
Rozpozná faktury v předmětu i těle emailu
Zachytí FA, FA/, FA12345, FA/2024/001
Přeskočí zálohové faktury a proformy
Stáhne PDF, JPG, PNG, TIFF přílohy
Uloží je do složky na Google Drive
Označí email štítkem „Faktura-stažena"
Nezduplikuje – každý email zpracuje jen jednou

KROK 3

Vložit kód agenta

Zkopíruj celý kód níže a vlož ho do prázdného souboru Code.gs. Poté uprav řádek s ID složky.

Code.gs — Gmail Faktura Agent 
// =============================================
// GMAIL FAKTURA AGENT
// Stahuje faktury z Gmailu na Google Drive
// Spouští se automaticky každou hodinu
// =============================================

// ⚙️ NASTAVENÍ — UPRAVTE PODLE SEBE
const CONFIG = {
  // ID vaší složky na Google Drive (viz Krok 1)
  driveFolderId: "VLOŽTE_ID_SLOŽKY_ZDE",

  // Název štítku v Gmailu (vytvoří se automaticky)
  gmailLabel: "Faktura-stažena",

  // Klíčová slova pro zachycení faktury
  keywords: [
    "faktura", "invoice", "rechnung", "faktúra",
    "daňový doklad", "vyúčtování služeb",
    "faktura přijata", "billing", "bill"
  ],

  // Regex pro FA, FA/, FA12345, FA/2024/001 atd.
  faPattern: /\bFA[\/\-]?[\d]{0,10}\b/i,

  // Výjimky — tyto emaily se přeskočí
  excludeKeywords: [
    "zálohová faktura", "záloha", "proforma",
    "advance invoice", "anzahlung", "deposit"
  ],

  // Povolené přípony souborů
  allowedExtensions: [".pdf", ".jpg", ".jpeg", ".png", ".tiff", ".tif"]
};

// =============================================
// HLAVNÍ FUNKCE — spouští se automaticky
// =============================================
function zpracujFaktury() {
  const label = getOrCreateLabel(CONFIG.gmailLabel);
  const folder = DriveApp.getFolderById(CONFIG.driveFolderId);
  const existingFiles = getExistingFileNames(folder);

  const keywordQuery = CONFIG.keywords
    .map(k => `"${k}"`).join(" OR ");
  const query = `has:attachment (${keywordQuery}) -label:${CONFIG.gmailLabel}`;
  const threads = GmailApp.search(query, 0, 50);
  let count = 0;

  for (const thread of threads) {
    const messages = thread.getMessages();

    for (const msg of messages) {
      const subject = msg.getSubject();
      const body = msg.getPlainBody();
      const combined = `${subject} ${body}`;
      const combinedLow = combined.toLowerCase();

      // Přeskočit zálohy
      const jeZaloha = CONFIG.excludeKeywords
        .some(k => combinedLow.includes(k.toLowerCase()));
      if (jeZaloha) continue;

      // Zkontrolovat klíčová slova nebo FA vzor
      const maKlicoveSlovo = CONFIG.keywords
        .some(k => combinedLow.includes(k.toLowerCase()));
      const maFaVzor = CONFIG.faPattern.test(combined);
      if (!maKlicoveSlovo && !maFaVzor) continue;

      // Zpracovat přílohy
      for (const att of msg.getAttachments()) {
        const name = att.getName().toLowerCase();
        const allowed = CONFIG.allowedExtensions.some(ext => name.endsWith(ext));
        if (!allowed) continue;

        const date = Utilities.formatDate(msg.getDate(), "Europe/Prague", "yyyy-MM-dd");
        const safeName = att.getName().replace(/[^a-zA-Z0-9.\-_čšžřáíéůúýóěňČŠŽŘÁÍÉŮÚÝÓĚŇ]/g, "_");
        const fileName = `${date}_${safeName}`;

        if (existingFiles.has(fileName)) continue;
        folder.createFile(att.copyBlob().setName(fileName));
        existingFiles.add(fileName);
        count++;
      }
    }
    thread.addLabel(label);
  }

  Logger.log(`✓ Hotovo. Uloženo souborů: ${count}`);
}

// =============================================
// POMOCNÉ FUNKCE
// =============================================
function getOrCreateLabel(name) {
  let label = GmailApp.getUserLabelByName(name);
  if (!label) label = GmailApp.createLabel(name);
  return label;
}

function getExistingFileNames(folder) {
  const set = new Set();
  const files = folder.getFiles();
  while (files.hasNext()) set.add(files.next().getName());
  return set;
}

KROK 5

Otestovat ručně

  1. Klikni na tlačítko ▶ Spustit nahoře v Apps Script
  2. Při prvním spuštění tě Google požádá o oprávnění – klikni Povolit
  3. Zkontroluj Protokoly (Zobrazit → Protokoly) – měl by tam být: "Hotovo. Uloženo souborů: X"
  4. Zkontroluj složku na Drive – soubory by měly být tam
ℹ️ Google označí script jako „neověřenou aplikaci" – to je u vlastních scriptů normální. Klikni Pokračovat a potvrď oprávnění.
KROK 6

Nastavit automatické spouštění každou hodinu

  1. Klikni vlevo na ikonu hodin ⏱ Triggery
  2. Klikni + Přidat trigger (vpravo dole)
  3. Nastav: Funkce → zpracujFaktury · Zdroj → Časový spínač · Interval → Každou hodinu
  4. Klikni Uložit
✅ Od teď agent běží každou hodinu sám. Každý zpracovaný email dostane štítek Faktura-stažena – takže vždy víš co už bylo uloženo a nic se nezduplikuje.
Tipy navíc

Doporučení pro lepší výsledky

🔍 Jak funguje rozpoznávání FA formátů

Kód používá regulární výraz /\bFA[\/\-]?[\d]{0,10}\b/i, který zachytí všechny běžné varianty: FA, FA/, FA2024001, FA/2024/001, FA-001. Funguje v předmětu i v těle zprávy.

⚠️ Samotné „FA" je krátká zkratka – sleduj prvních pár dní výsledky. Pokud ti agent občas zachytí i nerelevantní email, dej vědět – úprava trvá minutu.

📋 Kde najít chyby a logy

Apps Script ukládá záznamy každého spuštění. Najdeš je pod Triggery → Historie spuštění. Pokud agent nic nestáhne nebo spadne, uvidíš tam přesně proč – i s číslem řádku.

📁 Třídit do podsložek (volitelné)

Pokud budeš časem chtít faktury třídit do podsložek podle roku nebo měsíce, jde to přidat na cca 10 řádků kódu. Základ funguje i bez toho.
